## Welcome to Material React Table

A fully featured Material-UI V5 implementation of react-table, inspired by [material-table](https://material-table.com/) and the [MUI DataGrid](https://mui.com/components/data-grid/#mit-version), written from the ground up in TypeScript.

- The best mashup of Material-UI and React Table
- High quality and performant modern table features with a minimal amount of effort
- Easy to opt out of features or UI that you don't need
- Fully customizable with a variety of options
- Efficient bundle size (currently < 12kb gzipped)
