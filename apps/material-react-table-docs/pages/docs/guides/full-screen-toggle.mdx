### Disable Full Screen Toggle

The full screen toggle button is enabled by default. You can hide it by setting the `enableFullScreenToggle` prop to `false`.

```tsx
return (
  <MaterialReactTable
    columns={columns
    data={data}
    enableFullScreenToggle={false}
  />
);
```
